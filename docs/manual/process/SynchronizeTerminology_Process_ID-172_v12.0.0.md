# Process: Synchronize Terminology 

**[Created:** 18/11/2001 - **Updated:** 26/03/2008 **]**  
**Description:** Synchronize the terminology within the system.  
**Help:** Based on the entries in Window Element, the fields in windows, parameters, etc. are synchronized, if they are centrally maintained.  Also synchronizes Language Translation.
